# json_db


A database implemented by File system (json format)

## Performance Evaluation

| **Method** | **Time Complexity** |
| -----------| --------------------|
| Create     | O(n)                |
| Read       | O(1)                |
| Update     | O(n)                |
| Delete     | O(n)                |

- Isolation: ❌
- Durability: ✔️
